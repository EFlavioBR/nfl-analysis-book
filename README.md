# 🏈 nfl-analysis-book

Acompanhamento pessoal dos exercícios e análises do livro:

> **Football Analytics with Python & R: Learning Data Science Through the Lens of Sports**
> *Autores: Eric A. Eager e Richard Erickson*

Este repositório reúne as minhas resoluções e anotações ao longo da leitura, com foco em extrair e registrar os principais insights de cada capítulo.

Todo o conteúdo intelectual e didático pertence aos autores e à editora. Este material é de uso estritamente educacional e pessoal.
